# Portfólio — Tiago Vieira

Site pessoal e portfólio profissional de **Tiago Vieira**, desenvolvedor full stack e analista de sistemas.

**Site ao vivo:** [tiagovieira.dev.br](https://tiagovieira.dev.br)

## Sobre o projeto

Landing page e site institucional com foco em apresentação de serviços, projetos, habilidades técnicas, certificações e canal de contato. O layout é responsivo, com animações de scroll, menu mobile e otimizações de SEO.

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)
- Google Fonts (Inter e Poppins)
- Font Awesome (subset local — apenas os ícones utilizados)
- [Formspree](https://formspree.io/) — formulário de contato
- GitHub Pages — hospedagem

## Estrutura do repositório

```
Portifolio/
├── index.html              # Página principal (one-page com todas as seções)
├── CNAME                   # Domínio customizado (GitHub Pages)
├── assets/
│   ├── favicons/           # Ícones do site
│   └── fonts/              # Fontes subsetadas (Font Awesome)
├── components/
│   ├── menu.html           # Menu de navegação (carregado via fetch)
│   └── rodape.html         # Rodapé (carregado via fetch)
├── css/
│   ├── style.css           # Estilos principais
│   └── font-icons.css      # Ícones utilizados no projeto
├── js/
│   ├── bg.js               # Canvas de fundo animado
│   ├── fade.js             # Efeitos de revelação no scroll
│   └── menu-mobile.js      # Menu responsivo
└── pages/
    ├── servicos.html
    ├── projetos.html
    ├── sobre.html
    ├── habilidades.html
    ├── certificacoes.html
    └── contatos.html
```
